## 1. Requirements

* Ruby 1.8.7
* Rails 3.1
* MySQL 5.1
* MongoDB


## 2. Environment

Recommended environment is Debian 6 32-bit.


## 3. Installation

The following commands should be performed to install the product:
* git clone url
* cd specscan
* bundle install
* rake db:migrate
* rake test
* rails server


## 4. Usage

By default panel will be available at http://your-server:3000/
